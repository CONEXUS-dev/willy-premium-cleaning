## 🛠️ PART 2: HOW TO MAKE CHANGES

### The EASIEST Way (Recommended)
1. Go to your GitHub repo
2. Click the `public` folder
3. Click any file you want to change
4. Click the pencil icon ✏️ (top right)
5. Make your changes
6. Scroll down and click "Commit changes"
7. **Your website updates in 1-2 minutes!**

### What Each File Does:
- `index.html` = Your main homepage
- `404.html` = Error page (if someone types wrong address)
- `images/` folder = All your photos
- Other files = Website functionality (don't edit these)

## 📝 PART 3: COMMON UPDATES

### How to Change Your Phone Number
1. Go to GitHub → `public` folder
2. Click `index.html`
3. Click pencil ✏️ to edit
4. Find your phone number (use Ctrl+F to search)
5. Type your new number
6. Click "Commit changes"
7. **Done! Website updates in 2 minutes!**

### How to Update Services You Offer
1. Edit `index.html`
2. Find the services section
3. Add, remove, or change services
4. Click "Commit changes"

### How to Change Prices
1. Edit `index.html`
2. Find prices in service descriptions
3. Update your rates
4. Click "Commit changes"

### How to Add New Photos
1. Go to `public/images/` folder
2. Click "Upload files"
3. Select your photos from phone/computer
4. Upload them
5. **Important:** Use the same file names as old photos
6. Click "Commit changes"
