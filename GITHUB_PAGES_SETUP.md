# GitHub Pages Setup - 5 Minute Guide

Follow these steps to activate your FREE website hosting:

## 🚀 Activate GitHub Pages

1. **Go to your GitHub repo**: https://github.com/CONEXUS-dev/willy-premium-cleaning

2. **Click "Settings"** (top right, gear icon)

3. **Find "Pages"** in left menu (under "Code and automation")

4. **Configure Pages**:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/public`
   - Click **Save**

5. **Wait 2-3 minutes** while GitHub builds your site

6. **Visit your website** at the URL shown (usually: `https://CONEXUS-dev.github.io/willy-premium-cleaning/`)

## ✅ That's It!

Your website is now live and FREE forever!

## 🔄 How to Make Updates

### Quick Updates (No coding needed):
1. Go to your repo on GitHub
2. Click any file in the `public` folder
3. Click the pencil icon (edit)
4. Make your changes
5. Scroll down and click "Commit changes"
6. Website updates in 1-2 minutes!

### Replace Images:
1. Find the image in the `public/images` folder
2. Click "Upload files"
3. Drag your new image
4. Make sure it has the same name as the old one
5. Click "Commit changes"
